前端测试demo

#1 js框架为Vue.js

#1.1 安装vue cli ，建议安装3.x
$ npm install -g @vue/cli@3.0.4
# OR
$ yarn global add @vue/cli

#1.2 利用vue cli创建项目
$ vue create demo
#1.3 运行项目
$ cd demo
$ npm run serve


#2 UI框架为Ant Design

#2.1 安装antd
$ npm i ant-design-vue

#3 安装axios

#4 安装echart

#4.1 关于echart版本
安装5.x版本的echart时，提示在不到组件
建议安装4.x版本echart
https://blog.csdn.net/weixin_44874618/article/details/110647652
npm ls echarts  //查看echarts的版本
npm uninstall echarts //卸载
npm install echarts@4.9.0//重装
